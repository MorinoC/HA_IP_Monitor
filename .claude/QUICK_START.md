## 📖 3步快速上手

### 第1步: 阅读这3个文件 (5分钟)
```
1. README.md              → 了解项目是什么
2. PROJECT_STATUS.md      → 了解做到哪了
3. HANDOFF_最新日期.md    → 了解今天做什么
```

### 第2步: 开始编码 (根据交接文件)
- 按照优先级完成任务
- 遵循代码规范 (代码注释日语，用户交互中文)
- 频繁commit

### 第3步: 工作结束前 (10分钟)
```
1. 更新 PROJECT_STATUS.md
2. 更新 README.md
3. 删除旧交接文件
4. 创建新交接文件
5. Git commit & push
```

```
交接文件: HANDOFF_YYYY-MM-DD_vX.X.md
例如: HANDOFF_2025-11-13_v0.2.md
```
